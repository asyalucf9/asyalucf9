HTML конструювання

HTML (від англ. HyperText Markup Language - «мова гіпертекстової розмітки») - стандартизована мова програмування документів у Всесвітній павутині . Більшість веб-сторінок містять опис розмітки на мові HTML. Мова HTML інтерпретується браузерами; отриманий в результаті інтерпретації форматований текст відображається на екрані монітора комп'ютера або мобільного пристрою.

Версії

  • HTML 2.0 - опублікований IETF як RFC 1866 в статусі Proposed Standard ( 24 листопада 1995 року );
  • HTML 3.0 - 28 березень 1995 року - IETF Internet Draft (до 28 вересня 1995 года);
  • HTML 3.2 - 14 січня 1997 року ;
  • HTML 4.0 - 18 грудня 1997 року;
  • HTML 4.01 - 24 грудня 1999 року;
  • ISO / IEC 15445 2000 (так званий ISO HTML, заснований на HTML 4.01 Strict) - 15 травня 2000 року;
  • HTML5 - 28 жовтня 2014 року;
  • HTML 5.1 почав розроблятися 17 грудня 2012 року;
  • HTML 5.2 був представлений 14 грудня 2017 року;
    • HTML конструювання і сайтобудівництво для початківців і новачків
